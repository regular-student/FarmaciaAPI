Sobre o Projeto (C#/.NET)

Este projeto tem como proposta a criação de uma aplicação com o uso de C#, .Net Core aplicando o que foi mostrado nas aulas.

Criado por uma equipe de três alunos, o projeto implenta um site para um banco informações de remédios.

Funcionalidades

  • Estrutura CRUD: Implementação completa dos Metodos (GET, POST, PUT, DELETE).
  • Organização: Código modular e de fácil manutenção, seguindo padrões e boas praticas.
  • Demonstração: Frontend básico para teste e visualização das operações do backend.

Tecnologias

  • C#
  • Entity Framework
  • SQLite
  • HTML, CSS e JavaScript
